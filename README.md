# powerbi-superstore-sales-forecast-dashboard

# Project Overview

This project showcases an interactive **Sales Dashboard** built using Power BI to analyze business performance and generate **future sales forecasts**.
The dashboard provides meaningful insights into sales trends, profitability, and regional performance, helping in data-driven decision-making.

#Objectives

* Analyze historical sales data to identify patterns
* Compare performance across regions, categories, and segments
* Visualize key business metrics using interactive dashboards
* Generate **sales forecasts** to predict future trends

#Tools & Technologies

* Power BI
* Microsoft Excel
* Data Visualization Techniques
* Time Series Forecasting


# Dataset

* Name: Superstore Sales Dataset
* Format: Excel (.xlsx)
* Features Include:

  * Order Date
  * Sales
  * Profit
  * Region
  * Category
  * Customer Segment


# Dashboard Features

🔹 Sales Overview

* Total Sales, Profit, Quantity KPIs
* Summary cards for quick insights

🔹 Regional Analysis

* Sales distribution across regions
* Profit comparison

🔹 Category Analysis

* Performance by product category
* Identification of top-performing categories

🔹 Time Series Analysis

* Monthly and yearly trends
* Seasonal patterns

🔹  Forecasting

* Built-in Power BI forecasting used to predict future sales
* Helps in planning and decision-making
